# ft_printf
Project for 42 school

Simple reproduction of the printf function.

# Compilation

```
make
```

A libftprintf.a file will be created and added to your final compilation :

```
gcc *.c libftprintf.a
```

# Handled variable

- [x] %s
- [x] %d / %i

- [x] %p
- [x] %x
